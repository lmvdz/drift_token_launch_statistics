> this small program will let you view the claimed/forgone drift token launch statistics

`npm install`  

create `.env`  

set `RPC_URL` in `.env`  

`npm run start`


--- 
example output:

```txt
Forgone DRIFT 5,595,212.00
% Forgone 4.66%

Claimed DRIFT 100,485,764.00
% Claimed 83.74%

Unclaimed DRIFT 13,919,024.00
% Unclaimed 11.60%
```
